🔤 Word Chain Game

Word Chain Game is an interactive and educational web game where players build a chain of English words. Each new word must start with the last letter of the previous word, creating a continuous word chain 🔗.

The game helps improve vocabulary, spelling, and reaction time through fast-paced and enjoyable word challenges ⏱️🧠.

🎮 How It Works

1️⃣ Enter a valid English word.
2️⃣ The next word must begin with the last letter of the previous word.
3️⃣ Words cannot be repeated.
4️⃣ Keep the chain alive and beat your best score! 🔥

🛠️ Technologies Used

⚛️ React – For building a dynamic and interactive user interface
📦 an-array-of-english-words – Used as the English dictionary source
💅 Custom CSS – For styling and responsive layout

✨ Features

📝 Live feedback for each word (valid, invalid, already used)
📊 Score tracking and word history
🏆 Best score memory with congratulatory message
🔁 Restart option to instantly play again
⚡ Fast and smooth gameplay experience
